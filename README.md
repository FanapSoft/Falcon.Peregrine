# Falcon.Peregrine
An interface for real time streaming, cloud gaming and virtual desktop applications solutions powered by Fanap Falcon 


# Dependencies 
[Wolf.Engine libraries](https://github.com/WolfSource/Wolf.Engine)
- Wolf.System 
- Wolf.MediaCore
